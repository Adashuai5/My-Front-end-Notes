2.String

var s = new String('hello') 创建一个 String 对象

'hello' 与 new String('hello') 的区别是什么？看内存图

常用API


charAt()

String.charAt(n) 等价于 String[n]


charCodeAt()

String.charCodeAt() 可以打出对应字母的Unicode编码 


String.tiString()  

String.tiString()  转换整数为相应进制的字符串


String.trim()

String.trim() 可以去掉字符串两边的空格


String.concat()

String.concat() 可以连接两个字符串，相当于命令行中的 cat


String.slice() 有切开的意思，可以取到字符串中对应的内容

String.replace() 可以替换字符串中的相应字符
