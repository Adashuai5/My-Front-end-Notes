DOM：（xml）Document Object Model 把文档变成对象的模型

在没有 DOM 之前，JavaScript 想要操作页面（文档）内的元素比较麻烦，于是有了 DOM 把文档变成对象的模型，可以通过操作这些模型，直接操作元素，而不需要经过麻烦的遍历等基本操作，是 js 提供的 API 。

DOM 是一棵树（tree）

树上有 Node（节点）

Node 分为 Document（html）、Element（元素）和 Text（文本），以及其他不重要的。
